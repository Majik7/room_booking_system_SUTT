# room_booking_system_SUTT
Recruitment Task 1 for SUTT, By Akshat Ranjan

A simple command-line room booking system written in Python.
It allows users to create rooms, book hourly time slots, view details, and filter available rooms , saving and loading data from a CSV file.

## How to run
- Clone or download repositories
- Make sure you have Python installed
- Run the app using `python main.py`

## Requirements 
- No external modules are required
